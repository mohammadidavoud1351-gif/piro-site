# CLAUDE.md — پروژه سایت پیرو (PIRO Furniture)

این فایل راهنمای کار با کدبیس سایت پیرو برای Claude Code است.

---

## درباره پروژه

سایت فروش مبلمان لاکچری برند **پیرو (PIRO)** — نسخه اولیه (prototype) با HTML/CSS/JS خالص.
هدف: بازطراحی سایت فعلی (pirofurniture.com) با الهام از طراحی **Bolia.com** و هویت بصری اختصاصی پیرو.

- **مالک:** داود محمدی — مدیرعامل و سهامدار سابق برند پیرو
- **سبک برند:** مینیمال، جپندی + اسکاندیناوی، چوب طبیعی
- **مخاطب:** زنان ۲۵–۵۵ سال، آپارتمان‌نشین شهرهای بزرگ، درآمد بالا، دوستدار طبیعت
- **سایت واقعی:** https://pirofurniture.com/ (WordPress + WooCommerce)
- **اینستاگرام:** @piro_ir

## ساختار فایل‌ها

```
piro-site/
├── index.html       # صفحه خانه (Hero, دسته‌بندی موزائیک, محصولات ویژه, USP, خبرنامه)
├── shop.html        # فروشگاه (فیلتر سایدبار + گرید محصولات با داده JS)
├── product.html     # صفحه محصول (گالری, انتخاب رنگ چوب/سایز/تعداد, محصولات مرتبط)
├── about.html       # درباره ما (داستان برند, ارزش‌ها, آمار)
├── assets/
│   ├── styles.css   # استایل مشترک همه صفحات (متغیرها, هدر, فوتر, دکمه‌ها)
│   └── main.js      # اسکریپت مشترک (reveal, سبد, علاقه‌مندی, زبان, منوی موبایل)
└── CLAUDE.md        # این فایل
```

**نکته:** هر صفحه CSS اختصاصی خودش را در `<style>` داخل `<head>` دارد؛ استایل‌های مشترک در `assets/styles.css`.

## هویت بصری (Design Tokens)

تمام رنگ‌ها و فونت‌ها به‌صورت CSS variables در `assets/styles.css` (بخش `:root`) تعریف شده‌اند:

| متغیر | مقدار | کاربرد |
|-------|-------|--------|
| `--burgundy` | `#8B2635` | رنگ اصلی برند (قرمز زرشکی) |
| `--burgundy-dark` | `#6E1D29` | حالت hover |
| `--ink` | `#1A1A1A` | متن و دکمه تیره |
| `--bg` | `#FFFFFF` | پس‌زمینه اصلی |
| `--bg-warm` | `#F0EBE4` | بخش‌های کرم |
| `--wood-1/2/3` | رنگ‌های چوب | placeholder عکس‌ها |

**فونت‌ها:**
- لاتین/انگلیسی: `Jost` (کلاس `.lat`)
- فارسی: `Vazirmatn` (پیش‌فرض body)

**قاعده مهم:** همیشه از متغیرها استفاده کن، نه مقادیر hardcode. سایز فونت‌ها با `clamp()` ریسپانسیو هستند.

## نکات کلیدی برای ویرایش

1. **RTL/LTR:** سایت RTL است. متن‌های انگلیسی با کلاس `.lat` به LTR تبدیل می‌شوند.
2. **منوی مگا:** ساختار `.has-mega > .mega` — در هر ۴ صفحه تکرار شده. تغییر منو باید در همه صفحات اعمال شود.
3. **هدر و فوتر:** کد یکسان در هر ۴ فایل کپی شده (چون static HTML است). تغییر باید همگام باشد.
4. **عکس‌ها:** همه عکس‌ها فعلاً placeholder چوبی (کلاس `.wood`, `.wood.soft`, `.wood.deep`) هستند. هنگام افزودن عکس واقعی، این div‌ها با `<img>` جایگزین می‌شوند.
5. **داده محصولات shop:** در `shop.html` آرایه `products` در انتهای فایل تعریف شده و گرید را می‌سازد.

## اصول انتخاب عکس (مهم — همیشه رعایت شود)

برای صفحات اصلی سایت پیرو، عکس‌ها باید:
1. **دکوراتیو** باشند (صحنه چیدمان‌شده در فضای واقعی)، نه صرفاً صنعتی روی پس‌زمینه سفید
2. **کیفیت و موج چوب** را برجسته کنند — رگه، نقش و نگار و موج صفحه چوب باید کاملاً دیده شود
3. از **نمای درست و حرفه‌ای** عکاسی شده باشند
4. **رزولوشن بالا** (Hero ~3000px، کارت‌ها ~1500-1600px) با کیفیت ۸۸-۹۰٪
5. عکس‌های دکوراتیو معمولاً در ریشه پوشه محصول هستند (نام‌هایی مثل `_DSC33xx`, `_DSC57xx`, یا پسوند ` E` = ادیت‌شده)، نه در زیرپوشه `piro/` که عکس‌های استودیویی سفید است

ابزار بهینه‌سازی: `tools/resize.ps1` (System.Drawing — بدون نیاز به نصب).
نکته: pip/Pillow به دلیل تحریم در دسترس نیست؛ از System.Drawing استفاده شود.
